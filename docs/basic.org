#+title: Basic

Я так понял, что устройства замаплены не через mmu. То есть через физические адреса можно обращаться к устройствам.

Оперативная память тут начинается с адреса 0x80000000 и занимает ровно 2 ГиБ.

Aboot запускает ядро с выключенным mmu и кэшем. Но на уровне EL1. То есть, существуют инструкции, которые нашему коду вызывать запрещается.
Вместо этого нужно использовать инструкцию smc.
