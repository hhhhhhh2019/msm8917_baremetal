#+title: GIC

Судя по регистру GICD_IIDR тут используется не контроллер от arm, а специальный от qualcomm, но он (вроде как) должен быть совместим с arm gic v2.

Базовый адрес дистрибьютора = 0xb000000
Базовый адрес интерфейса процессора = 0xb002000

https://developer.arm.com/documentation/ihi0048/latest/

* Дистрибьютор
он нужен чтобы:
 + глобально переключать проброс прерываний на процессор
 + отдельно переключать прерывания
 + устанавливать приоритет прерываний
 + устанавливать нужный процессор для каждого прерывания
 + устанавливать группу для прерывания
 + пробрасывать SGI(что бы это ни было) на несколько процессоров
